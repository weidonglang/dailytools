# 源代码整理说明

## 源代码目录结构

- `tauri/src/`：前端 TypeScript 源码。
- `tauri/src-tauri/src/`：Rust 后端源码。
- `tauri/src-tauri/src/env_core/`：环境变量可靠性模块。
- `tauri/src-tauri/src/cleanup/`：C 盘急救模块。
- `tauri/src-tauri/src/safety/`：安全说明模块。
- `docs/`：说明文档、测试报告和软著材料。

## 主要源文件说明

- `lib.rs`：Tauri 命令注册和主要业务入口。
- `main.ts`：前端页面和事件入口。
- `env_core/*.rs`：环境快照、计划、验证、回滚。
- `cleanup/*.rs`：扫描、清理、搬家、重复文件、应用占用。
- `safety/*.rs`：风险等级和免责声明。

## 不纳入源代码打印的目录

- `target/`
- `node_modules/`
- `dist/`
- `release-assets-*`
- `.git/`

## 生成文件排除说明

构建缓存、依赖缓存、打包产物、临时文件、日志和测试临时目录不纳入源码打印。

## 构建产物排除说明

安装包、MSI、exe 和 SHA256 清单用于发布，不作为源码材料打印。
