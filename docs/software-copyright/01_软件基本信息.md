# 软件基本信息

- 软件名称：DevEnv Manager 开发环境诊断与安全配置管理软件
- 软件简称：DevEnv Manager
- 版本号：V1.0
- GitHub 对应版本：v1.5.1 Final Stable
- 开发语言：Rust、TypeScript
- 运行环境：Windows 10/11
- 开发完成日期：2026-06-27

## 主要功能

- 开发运行时安装、切换、验证和卸载。
- 用户级环境变量可靠性检查、修复、备份和恢复。
- Python pip/venv/ssl/sqlite3/ctypes/tkinter 完整性检查。
- Java、Nacos、Nexus、Maven、Gradle 环境验证。
- 项目启动向导、IDEA 配置只读分析、端口管理。
- C 盘急救、空间搬家、数据库服务检查和报告导出。

## 技术特点

- Tauri 2 桌面应用。
- Rust 后端执行系统级检查和受控操作。
- TypeScript 前端展示计划、风险说明和报告。
- 用户级环境变量修改具备预览、备份、验证和恢复流程。

## 适用用户

适合 Windows 上学习或维护 Java、Python、Node.js、Maven、Gradle、Go 等开发环境的个人用户、学生和开发者。
