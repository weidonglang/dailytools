# 用户操作手册

## 安装软件

从 GitHub Release 下载安装包，使用 SHA256 校验文件。

## 启动软件

打开 DevEnv Manager，先阅读使用前安全说明。

## 配置安装根目录

在总览页确认安装根目录，推荐使用非系统盘下的 `DevEnvManager` 目录。

## 安装 JDK

在版本管理页选择 JDK 发行版和版本，点击安装。安装后检查 `java`、`javac`、`jar` 和 `JAVA_HOME`。

## 安装 Python

在版本管理页选择 Python 版本，点击安装。安装后进行完整性检查。

## 检查环境变量

在环境页点击“检查可靠性”，查看 raw/expanded、PATH 和工具来源。

## 修复 Python pip

在 Python 环境分析中生成受管 Python pip 修复计划，确认后执行。

## 选择项目文件夹

在项目页点击“选择文件夹”，选择项目根目录后自动分析。

## 读取 IDEA 配置

点击“只读读取 IDEA 配置”，查看 Project SDK 和当前 `JAVA_HOME` 匹配情况。

## 验证 Nacos/Nexus Java 环境

选择 Nacos 或 Nexus 根目录后点击对应验证按钮。

## 导出报告

在诊断、清理或环境页面导出 Markdown/JSON 报告。

## 恢复环境变量

在环境备份中心选择备份记录，确认后恢复。
