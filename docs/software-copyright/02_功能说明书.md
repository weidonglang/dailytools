# 功能说明书

## 总览

展示当前生效的 Java、Python、Node.js、Maven、Gradle、Go、端口记录和 PATH 警告。

## 版本管理

支持受管 JDK、Python、Node.js、Maven、Gradle、Go 的下载、安装、切换、验证和卸载。

## 环境变量可靠性

读取当前进程环境和 Windows 用户环境，展示 `JAVA_HOME` raw/expanded、PATH 命中顺序、重复项、旧受管项和工具来源。

## Python 修复

检查 pip、venv、ssl、sqlite3、ctypes 和 tkinter。受管 Python 缺少 pip 时可生成修复计划。

## Java 验证

验证 `java.exe`、`javac.exe`、`jar.exe`、`JAVA_HOME`、PATH、Maven、Gradle，以及 Nacos/Nexus 等 Java 消费者环境。

## 项目启动向导

支持选择项目文件夹，识别项目类型，生成推荐命令和 VS Code/IDEA 配置预览。

## IDEA 配置读取

只读读取 `.idea/misc.xml`、`.idea/modules.xml`、`.idea/compiler.xml` 和 `*.iml`，不修改项目配置。

## 端口管理

扫描端口、展示进程来源、服务名、风险说明，并在确认后结束普通进程。

## C 盘急救

提供只读体检、清理计划、开发缓存官方命令、桌面/下载分析、大文件、重复文件、空间搬家和扩容检测。

## 数据库修复

只读检查 MySQL 服务、配置、Data 系统库和候选业务库；修复前要求计划和备份。

## 报告与回滚

支持导出诊断报告、清理报告、环境备份记录和空间搬家回滚记录。
