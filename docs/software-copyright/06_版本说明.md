# 版本说明

- 软件版本：V1.0
- GitHub 对应版本：v1.5.1 Final Stable
- 完成日期：2026-06-27

## 核心功能

- 开发环境诊断。
- 运行时安装和强验证。
- 环境变量可靠性修复。
- 项目启动向导。
- IDEA 配置只读分析。
- Java 消费者验证。
- C 盘急救与报告。

## 修复问题

- Python tkinter、pip、venv、ssl、sqlite3、ctypes 检查。
- 项目路径支持选择文件夹。
- IDEA 项目 JDK 与命令行 JDK 匹配诊断。
- 运行时安装后状态分型。
- Java 消费者环境读取解释。

## 稳定性说明

本版本不新增系统管家类方向，重点强化现有功能的验证、提示、报告、备份和确认。

## 不包含的功能

不包含注册表清理、驱动管理、杀毒扫描、隐私清理、自动关闭系统服务或自动修改系统级环境变量。
