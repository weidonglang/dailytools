# 设计说明书

## 系统架构

系统由 Tauri 2 桌面外壳、Rust 后端和 TypeScript 前端构成。

## 前端结构

前端负责页面渲染、输入校验、选择文件夹、风险说明、确认弹窗、报告展示和命令调用。

## 后端结构

后端负责运行时安装、文件下载、SHA256 校验、环境变量读写、路径保护、项目分析、端口扫描、清理计划、数据库诊断和报告生成。

## 核心模块

- `env_core`：环境可靠性、计划、验证、回滚。
- `cleanup`：C 盘急救和空间分析。
- `safety`：风险等级、说明和确认。
- `mysql_repair`：MySQL 修复中心。

## 核心数据结构

- 环境快照。
- 环境修复计划。
- Python 完整性报告。
- 运行时强验证报告。
- IDEA 项目报告。
- Java 消费者报告。
- 清理计划和清理结果。

## 运行时安装流程

查询元数据、选择下载源、校验 URL、下载、校验 SHA256、解压、目录检查、组件检查、登记、切换 current、验证环境。

## 环境变量修复流程

读取环境、生成快照、生成计划、展示 diff、备份、写入、广播、验证、报告。

## 项目分析流程

选择文件夹、检查路径、识别项目文件、读取 IDEA 配置、生成推荐运行时和固定操作。

## 安全控制流程

只读优先、计划预览、用户确认、路径保护、备份、回读验证和报告记录。
